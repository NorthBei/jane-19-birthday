## Development Instruction

### Enviroment

- Pure HTML, CSS, Javascript
- Font Awesome 4.6.3
- animate.css 3.5.0
- LigatureSymbols 2.11

### Getting Started

Super easy to start

### Project setup

Open vscode

### Start Development

Edit html,css,js file directly

### Lint all files

No any linting

### Build Production

No need to build

### Testing Built Result

Open `index.html` in browser to see the result
